Sobre este portfólio

Esta página demonstra domínio básico de estrutura HTML, estilização com CSS, responsividade, cards interativos, filtro por categoria e manipulação do DOM com JavaScript.

Objetivo do projeto

Criar uma página visualmente atrativa para apresentar informações organizadas sobre animes, usando uma estética inspirada em energia, ação, fantasia e cultura pop japonesa.

  • Layout responsivo para celular, tablet e computador.
  • Cards dinâmicos gerados pelo JavaScript.
  • Filtro por categoria: clássicos do Brasil e destaques de 2026.
  • Modal com informações extras ao clicar no card.

Recursos utilizados

HTML5 semântico
CSS3 com gradientes, animações e responsividade
JavaScript para renderização, filtros e modal
Sem bibliotecas externas
